# PM04-Software-Support
# ПМ.04 — Сопровождение и обслуживание программного обеспечения компьютерных систем

## 📌 Описание проекта

Данный репозиторий содержит все материалы, отчёты и результаты выполнения производственной практики по профессиональному модулю **ПМ.04** в рамках специальности **09.02.07 Информационные системы и программирование** (Слободской колледж педагогики и социальных отношений).

Практика направлена на закрепление навыков установки, настройки, сопровождения и защиты программного обеспечения, а также на освоение инструментов виртуализации, администрирования и систем контроля версий.

---

## 🧰 Что включает репозиторий

- **Отчёт по практике** (`Otchyot.docx`) — полный отчёт со скриншотами, таблицами и выводами.
- **Дневник практики** — сведения о выполненных работах с разбивкой по дням.
- **Аттестационный лист** — экспертная оценка руководителя.
- **Материалы по каждому пункту задания**:
  - Обоснование конфигурации
  - Установка ПО (1С, SSMS, Kaspersky, VirtualBox, Visual Studio, WPS Office)
  - Обеспечение доступа пользователям (Active Directory)
  - Обеспечение совместимости компонент
  - Контроль качества ПО
  - Анализ условий эксплуатации
  - Предложения по модификации ПО
  - Система контроля версий (Git, GitHub)
  - Защита программного обеспечения (Kaspersky, брандмауэр, UAC)
  - Тестирование ПО
- **Скриншоты** основных этапов работы.

---

## 🛠️ Технологии и инструменты

- **Операционная система**: Windows 10 Pro x64
- **Виртуализация**: Oracle VirtualBox 7.2.8
- **Серверные технологии**: Windows Server 2022, Active Directory
- **СУБД и администрирование**: Microsoft SQL Server Management Studio (SSMS)
- **Разработка и сборка**: Visual Studio 2026 Community, .NET 6
- **Open‑source проект**: Beginner Calculator (собран из исходников)
- **Защита ПО**: Kaspersky Standard, брандмауэр Windows, UAC
- **Контроль версий**: Git, GitHub
